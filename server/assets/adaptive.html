Фреймворк Bootstrap включает в себя лучшие практики для создания адаптивных веб-сайтов
На данной странице представлен пример хедера Navbar
Хедер уменьшает свой размер и сворачивает кнопки, в зависимости от размера экрана

Для создания похожего хедера пригодятся:

  • свойства CSS3
    display: flex
    flex-direction: row/column
    justify-content: flex-start
    @media (max-width: 124567890px) { ... }

  • vanillaJs, чтобы открыть или закрыть меню хедера
Подробное описание адаптивного CSS3
Плюсы:
  • меньше работы программистов, чем при раздельных сайтах для смартфонов и компьютеров
Минусы:
  • придётся выбрать mobile-first или desktop-first, при этом что-то пострадает в качестве
